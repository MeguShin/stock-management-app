# 開発言語
・Ruby 3.0.1

・Rails 6.0.3

# 実行手順

$ git clone git@github.com:MeguShin/stock-management-app.git

$ cd stock-management-app

$ bundle install

$ rails db:create && rails db:migrate

$ rails s
